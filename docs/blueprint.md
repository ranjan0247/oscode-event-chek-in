# **App Name**: RSVPZen

## Core Features:

- Excel Upload: Admin portal feature to upload an Excel sheet containing guest data.
- Data Migration to Firestore: Automatically migrate the guest data from the uploaded Excel sheet into Firestore.
- Homepage Guest List Display: Display the list of guests on the homepage, populated from Firestore data.
- Check-in Feature: Enable a check-in feature to mark guests as attended.
- Backend Attendance Tracking: Update guest attendance status in Firestore upon check-in.
- Real-time Frontend Updates: Reflect attendance status updates in real-time on the frontend.
- Admin Portal: Create a secured admin portal for backend task

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to create a calming and elegant feel.
- Background color: Light Gray (#F5F5F5), nearly white to provide a gentle, neutral backdrop.
- Accent color: Muted Blue (#B0C4DE) to highlight interactive elements, ensuring a focus without being too harsh.
- Font pairing: 'Belleza' (sans-serif) for headings and short amounts of text, and 'Alegreya' (serif) for body.
- Use elegant and simple line icons for RSVP actions.
- Maintain a clean, minimal layout with ample white space to enhance readability.
- Subtle transitions for guest check-in status changes.
